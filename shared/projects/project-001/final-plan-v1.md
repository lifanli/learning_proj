## 一、执行摘要

本项目旨在构建一个**多智能体协作的学习系统**，通过 AI 智能体（教师、助教、评估者等）提供个性化学习体验。

**核心特性**:
- 多智能体实时协作对话
- 个性化学习路径推荐
- 知识检索增强（RAG）
- 学习进度跟踪与分析

**技术栈**:
- 前端：React 18 + TypeScript + Vite
- 后端：待创建后端工程师负责
- 存储：MinIO（对象存储）+ pgvector（向量检索）
- RAG：LightRAG

**开发周期**: 5 周  
**当前状态**: 方案设计完成，准备进入开发阶段

## 三、技术选型决策

### 3.1 前端技术栈

| 决策点 | 选项 | 最终选择 | 决策理由 |
|--------|------|----------|----------|
| **框架** | React vs Vue | **React 18** | 团队熟悉度高，生态成熟 |
| **语言** | TypeScript vs JavaScript | **TypeScript** | 类型安全，利于维护 |
| **状态管理** | Zustand vs Redux | **Zustand** | 轻量简单，足够使用 |
| **UI 组件** | shadcn/ui vs Ant Design | **shadcn/ui** | 高度可定制，现代化设计 |
| **构建工具** | Vite vs Webpack | **Vite** | 开发速度快，配置简单 |
| **实时通信** | WebSocket vs SSE | **WebSocket + SSE** | 双向通信 + 流式响应 |

### 3.2 后端技术栈（待创建后端工程师）

| 决策点 | 选项 | 推荐选择 | 决策理由 |
|--------|------|----------|----------|
| **运行时** | Node.js vs Python | **Python** | AI/ML 生态好 |
| **框架** | FastAPI vs Flask | **FastAPI** | 性能好，自动文档 |
| **数据库** | PostgreSQL vs MongoDB | **PostgreSQL** | 关系型 + pgvector |
| **向量检索** | pgvector vs Chroma | **pgvector** | 与主数据库统一 |
| **对象存储** | MinIO vs S3 | **MinIO** | 自部署，成本低 |
| **RAG** | LightRAG vs LangChain | **LightRAG** | 轻量高效 |

### 3.3 基础设施

| 组件 | 技术选择 | 说明 |
|------|----------|------|
| **容器化** | Docker + Docker Compose | 简化部署 |
| **CI/CD** | GitHub Actions | 与代码仓库集成 |
| **监控** | Prometheus + Grafana | 开源成熟方案 |

## 四、系统架构

### 4.1 架构图

```
┌─────────────────────────────────────────────────────────┐
│                     用户界面                             │
│              (React + TypeScript)                        │
└────────────────────┬────────────────────────────────────┘
                     │ WebSocket/HTTPS
                     ↓
┌─────────────────────────────────────────────────────────┐
│                    API 网关                              │
│              (FastAPI + Auth)                           │
└────────────────────┬────────────────────────────────────┘
                     │
        ┌────────────┼────────────┐
        ↓            ↓            ↓
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  智能体服务  │ │  知识检索    │ │  用户服务    │
│  (多智能体)  │ │  (RAG)       │ │  (Auth/Profile)│
└──────┬───────┘ └──────┬───────┘ └──────┬───────┘
       │                │                │
       ↓                ↓                ↓
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│  PostgreSQL  │ │  MinIO       │ │  PostgreSQL  │
│  (pgvector)  │ │  (对象存储)   │ │  (用户数据)   │
└──────────────┘ └──────────────┘ └──────────────┘
```

### 4.2 核心模块

1. **前端模块**
   - 用户界面（React）
   - 智能体对话界面
   - 学习进度仪表盘

2. **智能体服务**
   - 主讲教师智能体
   - 助教智能体
   - 评估智能体
   - 搜索智能体

3. **知识检索**
   - LightRAG 实现
   - 向量检索（pgvector）
   - 文档存储（MinIO）

4. **用户服务**
   - 认证授权
   - 学习进度跟踪
   - 个性化推荐
