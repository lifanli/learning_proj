## 一、项目理解

### 1.1 核心目标
构建一个**多智能体协作的学习系统**，通过 AI 智能体提供个性化学习体验。

### 1.2 关键特性
- **多智能体协作**：教师、助教、评估者等角色协同工作
- **个性化学习**：根据用户水平调整内容和节奏
- **持续进化**：从用户反馈中学习改进
- **技术开放**：技术选型开放，MinIO/LightRAG 是推荐而非强制

### 5.3 关键技术实现

#### 智能体消息流式渲染
```typescript
const useAgentStream = (sessionId: string) => {
  const [messages, setMessages] = useState<Message[]>([]);
  
  useEffect(() => {
    const eventSource = new EventSource(`/api/sessions/${sessionId}/stream`);
    eventSource.onmessage = (event) => {
      const data = JSON.parse(event.data);
      setMessages(prev => [...prev, data]);
    };
    return () => eventSource.close();
  }, [sessionId]);
  
  return messages;
};
```

#### 多智能体上下文管理
```typescript
const useContextManager = () => {
  const contexts = useRef<Map<string, Message[]>>(new Map());
  
  const addMessage = (agentId: string, message: Message) => {
    const agentContext = contexts.current.get(agentId) || [];
    agentContext.push(message);
    if (agentContext.length > 20) agentContext.shift();
    contexts.current.set(agentId, agentContext);
  };
  
  return { addMessage, getContext: (id: string) => contexts.current.get(id) };
};
```

### 5.4 性能优化策略

| 优化点 | 方案 | 预期效果 |
|--------|------|----------|
| **首屏加载** | 代码分割 + 懒加载 | <2s |
| **智能体响应** | 流式渲染 + 乐观更新 | 感知延迟 <100ms |
| **大数据列表** | 虚拟滚动 | 支持 10000+ 消息 |
| **图片资源** | WebP + CDN + 懒加载 | 减少 70% 流量 |
| **离线体验** | Service Worker 缓存 | 离线可用核心功能 |
