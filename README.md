# 智能出版学习系统

> 智能体既是出版社（编写教材）又是作者（答疑互动）

**项目地址**: https://github.com/lifanli/learning_proj

---

## 📋 项目定位

降低跨平台学习成本，把所有学习内容（GitHub/微信/arXiv）集中到一个地方，通过智能体完成：
1. 收集 → 2. 整理 → 3. 内化 → 4. 反馈
